Windows setup

Follow along
step by step
Five steps. Don't skip ahead — each one needs the one before it. Look up at the screen if you get stuck. Steven and Arnel will walk the room.
1

Open PowerShell and install three apps

Click Start → type PowerShell → press Enter. Paste the lines below, all at once, and press Enter. Three things install: Windows Terminal (the modern terminal), Node.js (so we can install Claude Code in the next step), and Obsidian (the app that will read your vault).

winget install Microsoft.WindowsTerminal
winget install OpenJS.NodeJS.LTS
winget install Obsidian.Obsidian

If a UAC popup appears, click Yes. Each install takes ~30 seconds. When all three finish and you see the prompt again, you're done with this step.

2

Open Windows Terminal and install Claude Code

Close PowerShell. Open Windows Terminal from Start (it's the one you just installed). Paste this and press Enter:

npm install -g @anthropic-ai/claude-code

If you get npm.ps1 cannot be loaded because running scripts is disabled on this system, run this once, then retry:

Set-ExecutionPolicy -Scope CurrentUser RemoteSigned -Force

If it errors with "permission denied" or "EACCES", close Windows Terminal, right-click it in Start menu → Run as administrator → paste the same line again.

3

Make your folders on the Desktop

Use the names you wrote in the survey. Your AI's name is the parent folder. Your vault and your drop folder go inside it.

Open File Explorer → Desktop → right-click empty space → New > Folder. Make the parent folder first (your AI's name). Double-click into it. Make the two folders inside.

Example only: if you named your AI Sherlock, your vault Mindcastle, your drop folder Bakerstreet — you'd end up with Desktop\Sherlock\Mindcastle and Desktop\Sherlock\Bakerstreet. Use your names from the survey, not these.

4

Open your vault in ObsidianImportant

Obsidian is already installed (Step 1). Now you tell it which folder is your vault.

Open Obsidian from Start → click Open folder as vault → navigate to Desktop\YourAIName\YourVaultName (the inner one, not the parent) → Select Folder.

It will look empty. That's expected — your AI fills it in the next 2 minutes.

5

Open Windows Terminal inside your vault, then launch Claude

In File Explorer, navigate into your vault folder. Right-click empty space inside it → Open in Terminal. (Windows 10: hold Shift while right-clicking.)

In the Terminal that opens, paste this and press Enter:
claude --dangerously-skip-permissions

First launch: a browser tab opens for you to log in to your Claude Pro account. After login, come back to the terminal — it will show > and a blinking cursor. That's Claude. Look up at the screen — Steven will tell you what to paste next.

When you see > in your terminal — stop. Look up.
Steven will give you the master prompt to paste, and your AI builds itself from there.
